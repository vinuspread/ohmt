# Robotflow Style Guide & Design System

This design system is extracted directly from the official Robotflow style guide page.

## 1. Color Palette

### Neutral Colors
- **Neutral 100** (White Base): `#FFFFFF` (Used for pure white elements/cards)
- **Neutral 200** (Off-White Background): `#F8FAFF` (Light grey/blue tint for light layouts)
- **Neutral 300** (Light Border/Divider): `#F1F3F7`
- **Neutral 400** (Subtle Outlines): `#E1E4ED`
- **Neutral 500** (Muted Labels): `#B4B9C9`
- **Neutral 600** (Body Descriptions): `#6D758F`
- **Neutral 700** (Headers/Subheadings): `#353E5C`
- **Neutral 800** (Dark Theme Background/Text Ink): `#19213D` (Primary dark canvas)

### Brand Accent Color
- **Accent (Royal Indigo/Blue)**: `#3b82f6` (Used for primary CTA buttons, badges, status indicators)
